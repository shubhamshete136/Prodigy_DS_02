# PRODIGY_DS_02

## Task
Perform Data Cleaning and Exploratory Data Analysis (EDA) on Titanic Dataset.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Data Cleaning
- Handled missing values
- Removed Cabin column
- Removed duplicate records

## Exploratory Data Analysis
- Survival Analysis
- Gender vs Survival
- Passenger Class vs Survival
- Age Distribution
- Fare Distribution
- Correlation Heatmap

## Key Findings
- Female passengers had higher survival rates.
- First-class passengers survived more frequently.
- Most passengers were aged between 20–40 years.
- Passenger class and fare influenced survival.
